#Invoice-App-setup

For this project I have designed an application specifically for this automation task, You can find this application ready to download in my Invoice-Application repository.
or find it at https://github.com/CephasCh/Invoice-Application.git

When you download the zip file make sure you have it extracted in a permanent folder rather than downloads folder. 


In the extracted zip folder go to 'Release' to find the application


On initial startup you will be asked to choose or create a .xlsx file to save data into. Make sure you use the file you select here in the main project. 

<img width="685" height="432" alt="Screenshot 2026-05-02 055518" src="https://github.com/user-attachments/assets/ad978eb6-535e-4ef2-9209-1eb7c5449b0a" />


We have to make sure that the file path in the config file of your automation project and the selected .xlsx file on initial setup are the same


Also give your InvoiceEntryApp.exe location in the AppExePath (make sure you exclude the quotes while giving the location eg: projects\InvoiceEntryApp.exe✅  'projects\InvoiceEntryApp.exe'❌)
